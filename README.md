# PKI Express for PHP

Source code for the lacuna/pki-express composer package

The recommended way to install **PKI Express for PHP** is through [Composer](http://getcomposer.org):

    "require": {
        "lacuna/pkiexpress-client" : "^1.0.1"
    }
